# 领克第三方应用商店

这是一个为领克汽车用户开发的第三方应用商店前端项目，基于React + TypeScript + Tailwind CSS技术栈构建。

## 功能特性

- 应用下载管理（Free版和Pro版）
- 微信公众号关注与互动
- 支付支持功能
- 使用教程展示
- 响应式设计，适配多种设备

## 技术栈

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion (动画效果)

## 快速开始

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
pnpm dev
```

### 生产构建
```bash
pnpm build
```

### 预览生产版本
```bash
pnpm serve
```

## 项目结构

```
├── src
│   ├── components      # 公共组件
│   ├── hooks          # 自定义hooks
│   ├── lib            # 工具函数
│   ├── mock           # 模拟数据
│   ├── pages          # 页面组件
│   └── App.tsx        # 根组件
├── public             # 静态资源
└── vite.config.ts     # Vite配置
```

## 部署指南

1. 构建生产版本
2. 将dist目录内容上传到静态托管服务
3. 配置正确的base路径
